More Information

ESP32-C3-MINI-1 Wi-Fi® / BLE Module

Espressif Systems ESP32-C3-MINI-1 Wi-Fi® / BLE Module provides a turnkey wireless connectivity solution for Internet of Things (IoT), wearable electronics, and smart home applications. The ESP32-C3-MINI-1 is based on the ESP32-C3 single-core Wi-Fi and BLUETOOTH® LE 5.0 Microcontroller SoC (System on a Chip). This device includes an onboard PCB antenna and 4MB external SPI Flash memory.
